# Darkroom · 照片上傳 / 拍照介面

一個純前端的網頁介面,支援「拖放上傳照片」與「開啟鏡頭直接拍照」,拍好或上傳的照片會以底片沖印風格的卡片顯示在頁面下方。

**這是純前端展示 Demo**:照片只存在瀏覽器的記憶體中,重新整理頁面就會消失。之後要做「真的存檔」,可以再接後端(Node.js)或雲端服務(Firebase Storage / S3 等)。

## 檔案結構

```
photo-upload-site/
├── index.html   # 頁面結構(上傳區、鏡頭區、照片牆)
├── style.css    # 暗房/底片主題樣式
├── script.js    # 拖放上傳、選檔上傳、開鏡頭拍照的邏輯
└── README.md
```

## 本機預覽

不需要安裝任何東西,兩種方式都可以:

1. 直接雙擊 `index.html` 用瀏覽器打開(拖放上傳與選檔上傳都能用;但**拍照功能**在 `file://` 開啟時,瀏覽器可能會擋鏡頭權限)。
2. 想測試拍照功能,建議在資料夾內開一個本機伺服器,例如:
   ```bash
   cd photo-upload-site
   python3 -m http.server 8000
   ```
   然後瀏覽器打開 `http://localhost:8000`。

## 功能

- 拖放圖片到虛線框,或點擊「選取檔案」多選上傳
- 點「開啟鏡頭」呼叫裝置相機,按下快門圓鈕拍照
- 每張照片會以「沖印卡片」樣式滑入畫面,可個別移除
- 響應式版面,手機、桌面都能用
